# ML-Demo-Playground
A hands-on collection of **machine learning** and **deep learning** mini-projects ‚Äî from basic classifiers to advanced NLP, Vision Transformers, and vector databases.  
Each folder contains a self-contained Jupyter Notebook that demonstrates a specific concept or technique, making it easy to learn, run, and adapt for your own projects.
